# tong2-family

> 使用vue、vuex、vue-router、echarts搭建的一个班级数据展示平台
不支持IE8及其以下版本

## 运行步骤

``` bash
# 安装依赖
npm install

# 在本地启动服务，并且通过localhost:8080地址进行访问
npm run dev

# 编译并压缩代码便于发布
npm run build

# 运行单元测试
npm run unit
```
