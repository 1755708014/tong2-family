# family

> the statistics about tong2 family
不支持IE8及其以下版本

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```  

**IM_analyze**文件夹中是python写的分析群聊数据的相关代码
